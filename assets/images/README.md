# Images for Blog Posts

This directory contains images used in blog posts.

## Adding Images to Posts

1. **Upload images** to this `assets/images/` folder
2. **Reference in markdown** using: `![Alt text](assets/images/filename.png)`

## Recommended Image Types for Each Section

- **Hero/Logo images**: PNG with transparent background
- **Historical images**: JPG for photos, PNG for illustrations
- **Technical diagrams**: PNG or SVG for crisp diagrams
- **Charts/graphs**: PNG for complex visualizations

## Image Optimization

- Keep file sizes reasonable (< 500KB per image)
- Use descriptive filenames: `gerege-logo.png`, `silk-road-trade.jpg`
- Consider using WebP format for better compression
